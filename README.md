# Nostra - Fashion E-Commerce Website

Nostra is a modern and responsive fashion e-commerce website built using HTML, CSS, and JavaScript. The project focuses on providing a clean shopping experience with product filtering and search functionality.

## Live Demo

https://rajak1122.github.io/Nostra/

## Features

- Responsive Design
- Product Collection Page
- Product Search Functionality
- Category-Based Product Filtering
- Mobile Navigation Menu
- Modern Fashion UI
- Contact Page
- Smooth User Experience

## Technologies Used

- HTML5
- CSS3
- JavaScript
